# How to Embed Interactive Charts With Live Data (Step-by-Step)

We’ve released **embeddable interactive charts on Lychee**, making it possible to embed live charts directly into your website, blog, or documentation with just a single snippet.

Instead of exporting static images, you can now **embed interactive charts that stay live, dynamic, and fully explorable**.

This is useful if you want to:
- embed interactive charts in your website or blog post  
- share live data visualizations that update automatically  
- replace static images with dynamic, real-time charts  
- build dashboards with embeddable charts  

---

## Why Embed Interactive Charts Instead of Static Images

Most charts on the web are static screenshots. That creates a few problems:
- readers can’t inspect exact values  
- data becomes outdated instantly  
- charts lose context when shared  

With Lychee embeds, your charts stay:
- **interactive (hover tooltips, zoom, pan)**  
- **live (updates from the original dataset)**  
- **reusable (one chart, many embeds)**  

This turns every chart into a **live data experience instead of a static image**.

## How to Embed Interactive Charts on Any Website

You can embed Lychee charts on:
- WordPress  
- Webflow  
- Ghost  
- Framer  
- Wix  
- Notion (via embed blocks)  
- Custom HTML sites  

### Steps:

1. Open your chart in Lychee  
2. Go to **Export**  
3. Click **Share → Publish Embed**  
4. Copy the iframe snippet  
5. Paste it into your website’s HTML/embed block  
6. Publish  

That’s it — your chart is now an **interactive embeddable chart with live data**.

## Share URL vs Embed (When to Use Each)

### 🔗 Raw URL (Best for social sharing)
Example:
`https://lycheedata.com/<handle>/charts/<slug>`

Use this for:
- Twitter / X  
- LinkedIn  
- Slack  
- Discord  

It generates a preview card with your chart image.

---

### 📊 Embed (Best for websites & blogs)

Use iframe embeds when you want:
- interactive charts inside blog posts  
- live dashboards  
- data-rich landing pages  
- documentation pages  

This is the best way to **embed dynamic charts in a blog post or website**.

## FAQ

### What does it mean to embed a chart?
Embedding a chart means placing a live, interactive version of a chart directly inside a webpage using an embed snippet (usually iframe).

---

### How do I embed interactive charts in my website?
You generate an embed snippet in Lychee and paste it into any HTML or embed block on your website platform.

---

### Can I embed live data charts in a blog post?
Yes. Lychee charts update automatically, so embedded charts stay synced with the latest data.

---

### What is an embeddable chart?
An embeddable chart is a chart that can be inserted into websites or apps using a reusable embed link or iframe.

---

### Do embedded charts update automatically?
Yes. Charts stay connected to the original dataset and update dynamically.
