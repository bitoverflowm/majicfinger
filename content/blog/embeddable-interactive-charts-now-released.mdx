# Embeddable, Interactive Charts Now Released

We’ve released embeddable interactive charts on Lychee.

This means you can publish a chart to a public URL and embed it directly inside articles, guides, or other pages while keeping the chart interactive. Hover states, tooltips, and chart behavior are preserved, so readers can explore the data instead of seeing a static image.

Why this is useful:

- your readers can inspect exact values directly in your post
- your chart can stay up to date from the same saved chart source
- sharing is simpler because one chart URL can be reused across channels
